# Django Todo Application

A simple Todo application built using Django that supports user authentication and task management.  
Users can register, log in, manage their tasks, and securely log out with proper redirection.

## Project Overview

This project demonstrates the implementation of Django authentication along with a basic CRUD-based Todo system.  
It focuses on clean backend logic, proper URL handling, and simple UI integration.

## Features

- User registration
- User login and logout
- Logout redirects to custom login page
- Add, update, and delete todo tasks
- Authentication-protected pages
- Simple and clean interface

## Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite (default Django database)
